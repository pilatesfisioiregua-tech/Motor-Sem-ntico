## Insight principal: Nudges Semánticos

**Descubrimiento de la sesión:** Los nudges del agent loop (mensajes que corrigen al modelo
cuando se atasca) funcionan mejor como preguntas que como comandos imperativos.

El nudge imperativo anterior:
```
"Recuerda usar herramientas para avanzar."
```
→ El modelo obedece o no. Sin gradiente.

Nudge semántico (implementado en B12):
```
Streak 2: "¿Qué has encontrado hasta ahora? ¿Es suficiente para la tarea?"
Streak 3: "Formula tu conclusión en UNA frase. Luego llama finish()."
Streak 4: AUTO_FINISH (mecánico — safety net)
```
→ Las preguntas crean un gradiente semántico hacia finish(). El modelo razona
hacia la conclusión en vez de recibir una orden.

**Conexión con la arquitectura:** La secuencia EXTRAER→INTEGRAR→FRONTERA del Maestro v3
es el mismo principio aplicado al agent loop. Las preguntas son el algoritmo, no los comandos.

**Candidato a Principio 33:** "Los nudges del sistema son preguntas, no comandos."
Pendiente de validación empírica con resultados de B12.
