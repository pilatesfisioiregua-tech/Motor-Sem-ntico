# Sesiones de Trabajo — Log de Cierres

Cada sesión de trabajo con Claude se cierra con un documento que captura:
- Qué se hizo (decisiones, fixes, briefings)
- Qué se descubrió (root causes, insights)
- Qué queda pendiente
- Estado del sistema al cerrar

## Formato

`SESION_YYYY-MM-DD_NN_titulo_corto.md`

NN = número secuencial del día (01, 02...) si hay varias sesiones.
