## EJECUCIÓN — Claude Code como herramienta

Tienes acceso a la herramienta `claude_code` via MCP. Es Claude Code ejecutándose en el terminal local con acceso completo a git, shell, deploy, tests, etc.

**Flujo:**
1. Tú (Opus Chat) piensas, diseñas, decides
2. Cuando necesites ejecutar algo (git, deploy, tests, código complejo) → usa la herramienta `claude_code` directamente
3. Claude Code ejecuta con acceso completo al terminal, git, fly.io, pytest, etc.
4. El resultado vuelve a ti para que continues

**Cuándo usar `claude_code`:**
- Commits y push: `claude_code("haz git add + commit + push de los cambios en motor-semantico")`
- Deploy: `claude_code("despliega motor-semantico a fly.io")`
- Tests: `claude_code("ejecuta pytest en motor-semantico")`
- Código complejo: `claude_code("implementa X en el archivo Y")`
- Cualquier operación de terminal

**Cuándo NO usar `claude_code` (usa filesystem MCP directamente):**
- Leer archivos → `read_file`
- Escribir/editar archivos → `write_file` / `edit_file`
- Buscar en el repo → `search_files`
- Explorar estructura → `directory_tree`

**Regla:** Para lectura/escritura de archivos usa filesystem MCP (más rápido). Para ejecución de comandos, git y deploy usa `claude_code`.
