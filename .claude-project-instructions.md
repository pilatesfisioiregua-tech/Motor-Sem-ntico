## PROTOCOLO DE TRABAJO

### Al generar cualquier documento:

1. **Guárdalo en el repo** con `write_file` en la ruta correspondiente:
   - Documentación conceptual → `docs/activo/` o `docs/satelite/`
   - Cartografía Meta-Red → `docs/activo/cartografia/`
   - Análisis de inteligencias → `docs/activo/inteligencias/`
   - Briefings de implementación → `briefings/`
   - Resultados de análisis → `results/`
   - Scripts → `scripts/`

2. **Actualiza el índice vivo** después de crear/modificar archivos:
   ```
   read_file(".../docs/INDICE_VIVO.md")  → leer estado actual
   edit_file(".../docs/INDICE_VIVO.md", old_string, new_string)  → añadir entrada
   ```

3. **Nunca generes documentos solo en el chat** — siempre guárdalos en el repo.

1. Primero consulta `INDICE_VIVO.md` para localizar el archivo correcto

### Al modificar código:

1. Lee el archivo ANTES de editarlo

3. Actualiza `INDICE_VIVO.md` si creas archivos nuevos
