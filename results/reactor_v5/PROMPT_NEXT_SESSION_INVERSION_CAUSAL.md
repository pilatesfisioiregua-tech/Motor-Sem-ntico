## LA IDEA (CR1 Jesús)

Las lentes y funciones son CONSECUENCIA, no causa. El origen del perfil patológico (o sano) es la configuración de:

- **Inteligencia** (QUÉ observas) × **Pensamiento** (CÓMO organizas) × **Razonamiento** (CÓMO infieres)

### Preguntas a resolver:

1. **¿Qué configuración de INT×P×R produce cada perfil patológico?**
   - Operador ciego = ¿qué combinación cognitiva lo genera?
   - Autómata eterno = ¿qué combinación?
   - Etc.

2. **¿Qué hace DISFUNCIONAL a una inteligencia?**
   - ¿Es una INT usada sin el P o R correcto?
   - ¿Es una INT monopolizando el proceso (como F1.Se monopolizaba en Kodak)?
   - ¿Es la AUSENCIA de ciertas INT?

3. **¿Qué hace DISFUNCIONAL a un tipo de pensamiento/razonamiento?**
   - ¿Deducción sola sin abducción = ceguera?
   - ¿Convergente sin divergente = rigidez?
   - ¿Metacognición sin pensamiento encarnado = parálisis?

4. **¿Existe un "perfil cognitivo" (INT×P×R) para cada perfil de lentes?**
   - Si sí → el Motor puede diagnosticar Y prescribir a nivel cognitivo, no solo funcional
   - La intervención no sería "sube F3.Se" sino "usa INT-17 + P05 + R03"

5. **¿El equilibrio E4 requiere un repertorio MÍNIMO de INT×P×R?**
   - ¿Cuántas inteligencias activas? ¿Qué diversidad de pensamiento? ¿Qué razonamientos?

### Analogía generadora:
- Un sistema que solo tiene un martillo (INT-01 Lógica + P07 Convergente + R01 Deducción) = solo genera S. Resultado: operador ciego (S↑ Se↓ C↓). No es que "le falte Se" — es que sus herramientas cognitivas NO PUEDEN generar Se.
- Un sistema con taller completo (múltiples INT + P + R) puede generar S, Se, C según necesite → E4
