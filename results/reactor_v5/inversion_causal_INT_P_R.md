Hasta ahora el Motor diagnostica en lentes y funciones: "este sistema tiene S↑ Se↓ C↓, necesita F3.Se y F5.Se". Eso es correcto pero es como decir "el paciente tiene fiebre, hay que bajar la temperatura". La pregunta causal es: ¿qué PRODUCE la fiebre?

```
CAUSA:         INT × P × R (configuración cognitiva)

                    ▼
MECANISMO:     Cada combinación genera preferentemente ciertas lentes
                    │
                    ▼
CONSECUENCIA:  Perfil de lentes (S, Se, C) → perfil de funciones (7F × 3L)
```

Si el Motor puede diagnosticar a nivel de INT×P×R, la intervención no es "sube F3.Se" (síntoma) sino "activa INT-17 + P05 + R03" (causa).

## 2. INVERSIÓN: CONFIGURACIÓN COGNITIVA → PERFIL PATOLÓGICO

### 2.1 Principio general

Un perfil patológico emerge cuando el repertorio cognitivo del sistema está SESGADO hacia una lente y CARECE de herramientas para las demás.

```
REGLA IC1: Perfil de lentes = f(distribución de INT×P×R por lente)

Si >70% de las herramientas cognitivas activas generan la misma lente → esa lente domina.
Si <15% de las herramientas generan una lente → esa lente colapsa.
```

#### OPERADOR CIEGO (S↑ Se↓ C↓)

**Configuración cognitiva generadora:**

| Dimensión | Activas | Ausentes/atrofiadas |
|-----------|---------|---------------------|
| INT | 01(Lógica), 02(Computacional), 05(Estratégica), 07(Financiera), 16(Constructiva) | 17(Existencial), 18(Contemplativa), 03(Estructural), 08(Social) |

**Mecanismo causal:** El sistema tiene un taller lleno de herramientas de ejecución. Sabe calcular (INT-01), optimizar (INT-05), construir (INT-16), medir retorno (INT-07). Organiza convergentemente (P07) y deduce correctamente (R01). Todo esto genera S.

Pero NO tiene herramientas de cuestionamiento. No se pregunta POR QUÉ (INT-17 ausente). No contempla alternativas (INT-18 ausente). No busca explicaciones profundas (R03 ausente). No confronta opuestos (R08 ausente). No observa su propio proceso (P08 ausente).

**¿Por qué no genera Se?** Porque NINGUNA de las herramientas activas lo hace. La deducción (R01) extrae lo implícito — no genera comprensión nueva. El pensamiento convergente (P07) selecciona — no cuestiona. INT-01 encuentra contradicciones formales — no pregunta si las premisas merecen existir.

**¿Por qué no genera C?** Porque la continuidad requiere F7.Se (enseñar POR QUÉ, no solo CÓMO). Sin Se, la replicación transmite S sin sentido → autómata eterno si persiste.

**Ejemplo Maginot:** INT-01(deducción impecable) + INT-05(estrategia de línea fija) + P07(convergente: una solución) + R01(si guerra = trincheras, entonces muro) = S↑↑. Pero INT-13(Prospectiva) ausente, P09(Prospectivo) ausente, R06(Contrafactual: ¿y si la guerra no es estática?) ausente = Se↓↓.

**La causa real:** No es que "le falte Se". Es que sus herramientas cognitivas son INCAPACES de generar Se. Es como pedir a un microscopio que mida temperatura.

## 3. PREGUNTA 2: ¿QUÉ HACE DISFUNCIONAL A UNA INTELIGENCIA?

Una inteligencia no es disfuncional en sí misma. Se vuelve disfuncional por tres mecanismos:

### 3.1 Monopolio (una INT domina sin contrapeso)

```
INT-01(Lógica) sola, sin INT-17(Existencial) ni INT-08(Social)
= Formalismo sin sentido. Deduce correctamente desde premisas que nadie cuestiona.
= F3 formal sin F3.Se.
```

**Regla IC2:** Una inteligencia se vuelve disfuncional cuando opera en MONOPOLIO — sin las inteligencias complementarias que la equilibran.

| INT en monopolio | Genera | Carece de | Resultado |
|-----------------|--------|-----------|-----------|
| INT-01 sola | S (formalización) | Se (¿merecen las premisas?) | Maginot |
| INT-07 sola | S (rentabilidad) | Se (¿a qué coste humano?) | Boeing MAX |
| INT-17 sola | Se (cuestionamiento) | S (¿y qué hago?) | Parálisis existencial |
| INT-16 sola | S (prototipado) | Se (¿merece existir?) | Iteración sin dirección |
| INT-13 sola | C (proyección) | Se (¿proyecto DESDE QUÉ premisas?) | Extrapolación ciega |

### 3.2 Desacople INT-P (inteligencia sin el pensamiento correcto)

```
INT-17(Existencial) + P07(Convergente)
= "¿Cuál es el sentido de la vida? → selecciona UNA respuesta rápido"
= Se profunda ABORTADA por la estrategia de procesamiento.
```

**Regla IC3:** Una inteligencia se vuelve disfuncional cuando se procesa con un tipo de pensamiento incompatible.

| INT | P incompatible | Efecto |
|-----|---------------|--------|
| INT-17(Existencial) | P07(Convergente) | Trivializa la pregunta |
| INT-14(Divergente) | P07(Convergente) | La divergencia se cierra antes de explorar |
| INT-16(Constructiva) | P08(Metacognición) | Se piensa sobre construir en vez de construir |
| INT-10(Cinestésica) | P02(Sistémico) | Se analiza el movimiento en vez de moverse |
| INT-01(Lógica) | P06(Divergente) | Se generan muchas opciones sin rigor formal |

### 3.3 Desacople INT-R (inteligencia sin el razonamiento correcto)

```
INT-03(Estructural) + R01(Deducción)
= "Las premisas dicen que la identidad es X → la identidad es X"
= Nunca descubre la identidad REAL (necesita R03 Abducción)
```

**Regla IC4:** Una inteligencia se vuelve disfuncional cuando su mecanismo inferencial no puede generar el tipo de conclusión que la inteligencia necesita.

| INT | R incompatible | R necesario | Por qué |
|-----|---------------|-------------|---------|
| INT-03(Estructural) | R01(Deducción) | R03(Abducción) | El gap id↔ir se DESCUBRE, no se deduce |
| INT-08(Social) | R01(Deducción) | R03(Abducción) | La vergüenza oculta se abduce de señales |
| INT-13(Prospectiva) | R01(Deducción) | R06(Contrafactual) | El futuro se explora por contrafactuales |
| INT-15(Estética) | R07(Bayesiano) | R10(Retroductivo) | La belleza no se probabiliza, se retroduce |
| INT-05(Estratégica) | R03(Abducción) | R01(Deducción) | La secuencia de movimientos se deduce |

## 4. PREGUNTA 3: ¿QUÉ HACE DISFUNCIONAL A UN P O R?

### 4.1 P disfuncional: pensamiento sin complemento

La LEY TP3 de la taxonomía ya lo dice: S=HACER, Se=CUESTIONAR, C=ENSEÑAR. Un pensamiento se vuelve disfuncional cuando opera SIN su complemento natural:

| P activo | Complemento necesario | Sin complemento = |
|---------|----------------------|-------------------|
| P06(Divergente) | P07(Convergente) | Generación infinita sin cierre = visionario |
| P07(Convergente) | P06(Divergente) | Cierre prematuro sin exploración = operador ciego |
| P05(Primeros principios) | P04(Diseño) o P11(Encarnado) | Deconstrucción sin reconstrucción = parálisis |
| P08(Metacognición) | P11(Encarnado) o P13(Computacional) | Pensar sobre pensar sin actuar = recursión infinita |
| P09(Prospectivo) | P03(Crítico) | Proyección sin cuestionamiento de premisas = extrapolación ciega |

**Regla IC5:** Los tipos de pensamiento operan en PARES complementarios. Uno sin el otro = disfunción.

### 4.2 R disfuncional: razonamiento sin validación cruzada

| R activo | Validación necesaria | Sin validación = |
|---------|---------------------|------------------|
| R01(Deducción) sola | R02(Inducción) o R03(Abducción) | Certeza desde premisas no validadas = Maginot |
| R02(Inducción) sola | R06(Contrafactual) | Generalización sin testear excepciones = cisne negro |
| R03(Abducción) sola | R01(Deducción) | Explicaciones plausibles sin verificación formal |
| R04(Analogía) sola | R09(Eliminación) | Transferencia superficial sin depurar diferencias |
| R07(Bayesiano) con priors fijos | R08(Dialéctico) | Actualización que confirma sesgos = echo chamber |

**Regla IC6:** Los razonamientos se validan CRUZADAMENTE. Un razonamiento aislado amplifica sus propios sesgos.

**Esto permite al Motor hacer dos cosas nuevas:**

## 6. PREGUNTA 5: ¿E4 REQUIERE UN REPERTORIO MÍNIMO DE INT×P×R?

**Sí.** El análisis de los cierres del dataset (TCP/IP, Mercadona, Toyota, KonMari, DPP) revela un patrón:

### 6.1 Requisitos mínimos para E4

**Inteligencias:** Mínimo 3 de cada lente activas simultáneamente.

```
≥3 INT de S:   al menos 3 de {01, 02, 05, 07, 10, 11, 16}
≥3 INT de Se:  al menos 3 de {03, 04, 06, 08, 09, 12, 14, 15, 17, 18}
≥1 INT de C:   INT-13(Prospectiva) + al menos 1 secundaria de C {02, 04, 12, 16, 18}
```

**Pensamientos:** Al menos 1 par complementario activo + metacognición.

```
OBLIGATORIO:  P07(Convergente) + P06(Divergente)  — el par generación/selección
OBLIGATORIO:  P08(Metacognición)  — supervisión del proceso
DESEABLE:     P02(Sistémico) + P05(Primeros principios) + P09(Prospectivo)
```

**Razonamientos:** Al menos 1 de cada tipo funcional.

```
OBLIGATORIO:  R01(Deducción) O R07(Bayesiano)  — operatividad
OBLIGATORIO:  R03(Abducción)  — diagnóstico/explicación
OBLIGATORIO:  R04(Analogía) O R12(Transductivo)  — transferencia
DESEABLE:     R06(Contrafactual) + R08(Dialéctico)  — robustez
```

### 6.2 Repertorio mínimo E4 cuantificado

```
INTELIGENCIAS:  ≥7 activas (3S + 3Se + 1C mínimo)  de 18 posibles = 39%
PENSAMIENTOS:   ≥4 activos (P06+P07+P08 + 1 de Se)  de 15 posibles = 27%
RAZONAMIENTOS:  ≥3 activos (1S + R03 + 1C)           de 12 posibles = 25%
```

**Regla IC7:** E4 requiere un repertorio DIVERSO, no grande. No necesitas las 18 inteligencias — necesitas al menos 7 bien distribuidas entre las 3 lentes. La diversidad cognitiva es más importante que la cantidad.

| Potencial dormido | 5-7 | Se+C, S=0 | Incompleta |
| **E4 sano** | **≥7** | **S+Se+C** | **Alta y equilibrada** |

### 7.1 Implicación para el Motor vN

El pipeline actual diagnostica en Nivel 3-4 (lentes y funciones) y prescribe en Nivel 3 ("sube Se"). Con la inversión causal, el Motor puede:

1. **Diagnosticar en Nivel 1:** "El sistema usa estas INT, estos P, estos R"
2. **Explicar en Nivel 2:** "Esa configuración genera este sesgo de lentes"
3. **Prescribir en Nivel 1:** "Activa estas INT×P×R específicas"
4. **Verificar en Nivel 3-4:** "¿Se movieron las lentes y funciones?"

### 7.2 Pipeline actualizado con inversión causal

```
PASO 0:  Perfil de lentes (S_avg, Se_avg, C_avg) — DIAGNÓSTICO RÁPIDO
PASO 0.5: Estado de equilibrio (E1-E4) o perfil de desequilibrio

PASO 1: ★NUEVO★ DIAGNÓSTICO COGNITIVO
  - ¿Qué INT están activas/atrofiadas en el sistema?
  - ¿Qué P usa el sistema por defecto?
  - ¿Qué R emplea el sistema habitualmente?
  - Calcular: distribución INT×P×R por lente
  - Verificar: ¿hay monopolio? ¿hay desacoples INT-P o INT-R?

PASO 2: PRESCRIPCIÓN COGNITIVA (no funcional)
  - Según perfil → seleccionar INT×P×R que generan la lente faltante
  - Verificar que no hay incompatibilidades (IC3, IC4)
  - Verificar pares complementarios (IC5) y validación cruzada (IC6)

PASO 3: SECUENCIA COGNITIVA
  - Ordenar activación: Se primero (INT de Se + P de cuestionamiento + R de explicación)
  - Luego S (INT de acción + P de ejecución + R operativos)
  - Luego C (INT de transferencia + P de codificación + R de analogía)

PASO 4: EJECUCIÓN — las preguntas del programa compilado
  - Las INT seleccionadas generan las PREGUNTAS
  - Los P seleccionados organizan el PROCESO de responderlas
  - Los R seleccionados determinan CÓMO se responden

PASO 5: EVALUACIÓN
  - ¿Se movió la lente faltante?
  - ¿El repertorio cognitivo se amplió?
  - ¿El gap disminuyó?
  - ¿Emergió algún desacople nuevo?
```

## 8. LEYES DE LA INVERSIÓN CAUSAL

| # | Ley | Formulación |
|---|-----|-------------|
| IC1 | Perfil = distribución de repertorio | El perfil de lentes es función directa de la distribución INT×P×R por lente |
| IC2 | Monopolio = disfunción | Una INT en monopolio (sin complementarias) produce patología |
| IC3 | Desacople INT-P = abortamiento | Una INT procesada con P incompatible pierde su capacidad generadora |
| IC4 | Desacople INT-R = conclusión incorrecta | Una INT con R incompatible genera conclusiones que no corresponden al dominio |
| IC5 | P en pares complementarios | Los pensamientos son funcionales solo en pares (divergente+convergente, meta+encarnado, etc.) |
| IC6 | R se validan cruzadamente | Un razonamiento aislado amplifica sus propios sesgos |
| IC7 | E4 = diversidad mínima | El equilibrio requiere ≥7 INT + ≥4 P + ≥3 R distribuidos entre las 3 lentes |
